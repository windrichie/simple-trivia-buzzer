# Buzzer Sound Files

This directory should contain buzzer sound audio files in both MP3 and OGG formats for browser compatibility.

## Required Files

Each sound should have both formats:
- classic.mp3 / classic.ogg
- horn.mp3 / horn.ogg
- bell.mp3 / bell.ogg
- boing.mp3 / boing.ogg
- chime.mp3 / chime.ogg
- whoosh.mp3 / whoosh.ogg
- beep.mp3 / beep.ogg
- ding.mp3 / ding.ogg
- buzz.mp3 / buzz.ogg
- whistle.mp3 / whistle.ogg

## Recommended Sources

1. **Free Sound Libraries:**
   - Freesound.org (CC0 or CC-BY licensed sounds)
   - Zapsplat.com (free tier available)
   - Mixkit.co (free sound effects)

2. **Sound Characteristics:**
   - Duration: 0.5-1.5 seconds
   - Format: 44.1kHz sample rate
   - Size: Keep under 50KB per file for fast loading

3. **Search Terms:**
   - "game show buzzer"
   - "quiz button sound"
   - "notification sound"
   - Use the actual sound name (e.g., "bell sound", "horn sound")

## For Development

For testing purposes, you can temporarily use placeholder sounds or browser-generated audio until proper sound files are added.
